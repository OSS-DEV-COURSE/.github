## 📦 Organization 소개

이 Organization은 **오픈소스 소프트웨어 개발** 강의에서 진행하는
실습 및 프로젝트들을 정리하고 관리하기 위해 생성되었습니다.

## 🎯 목적

* 강의에서 진행한 실습 코드 및 결과물 관리
* 프로젝트별 코드 이력 관리 및 협업 경험
* GitHub 기반의 협업 및 오픈소스 개발 방식 학습

## 📁 구성

* 실습 내용은 주차별 또는 주제별로 Repository를 나누어 관리합니다.
* 프로젝트는 별도의 Repository로 분리하여 관리합니다.

## 🤝 운영 방식

* 각 Repository는 Git 기반으로 관리됩니다.
* Issue, Pull Request 등을 활용하여 협업을 진행합니다.
* 코드 리뷰 및 피드백을 통해 개선 과정을 경험합니다.

## 💬 기타

* 학습 과정에서 발생한 질문이나 고민은 자유롭게 공유합니다.
* 오픈소스 개발 방식에 익숙해지는 것을 목표로 합니다.
